# 桌面便签

简约美观的 Windows 桌面悬浮便签，支持工作记录、计划安排、密码备忘。

## 功能

- **悬浮桌面** — 无边框透明窗口，可置顶显示
- **收起到边缘** — 一键缩到屏幕右侧，点击即可展开
- **透明度调节** — 底部滑块实时调整窗口透明度
- **三个分类** — 工作 / 计划 / 密码，内容自动本地保存
- **系统托盘** — 关闭后最小化到托盘，双击恢复

## 运行

```bash
npm install
npm start
```

## 快捷操作

| 操作 | 说明 |
|------|------|
| 拖动标题栏 | 移动窗口 |
| 📌 | 切换置顶 |
| ◀ | 收起到屏幕边缘 |
| ✕ | 隐藏到系统托盘 |
| 透明度滑块 | 调节窗口透明度 |

## 打包

```bash
npm install
npm run build          # 安装程序（推荐）
npm run build:portable # 便携版 exe
```

安装程序输出：`dist/桌面便签-Setup-1.0.0.exe`

- 支持自定义安装目录
- 自动创建桌面和开始菜单快捷方式
- 默认开启**开机自启动**（托盘右键可关闭）
- 开机启动时静默到系统托盘，不弹窗口

便携版输出：`dist/桌面便签-1.0.0.exe`（双击即用，无需安装）

### macOS 安装包

> **注意：** macOS 的 `.dmg` 安装包只能在 Mac 电脑上打包，Windows 无法直接生成。

**在 Mac 上本地打包：**

```bash
npm install
npm run build:mac
```

输出：`dist/桌面便签-1.0.0-x64.dmg` 和 `dist/桌面便签-1.0.0-arm64.dmg`（Intel + Apple 芯片）

**在 Windows 上通过 GitHub Actions 打包：**

1. 将项目推送到 GitHub
2. 打开仓库 → Actions → 「Build macOS DMG」→ Run workflow
3. 完成后在 Artifacts 中下载 `.dmg` 文件

首次在 Mac 上打开时，若提示「无法验证开发者」，请在「系统设置 → 隐私与安全性」中点击「仍要打开」。
